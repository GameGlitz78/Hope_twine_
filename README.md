A narrative driven stealth prototype focused on player choices and system illusion. Set within a high security facility, the game combines environmental navigation and lightweight puzzle mechanics, including laser grids and pattern locks, to create a illusion of decision making under pressure.

The core design pillar explores the illusion of choice, where branching paths, interactive systems, and player inputs are designed to feel interactive and deceiving while ending towards a single end. Each interaction is perfected by player feedback of every playtesting iteration.

Built in Twine using Harlowe, the project puts importance in rapid prototyping, branching narrative design, and story driven puzzles, taking advantage of minimal systems to deliver a focused gameplay loop with strong thematic cohesion.
